Hello world!!

API services:

  • GET: all colors:

    http://localhost:5000/colors

  • GET: one colors:

    http://localhost:5000/color/1

  • GET: Filter by favorite/non-favorite:

    http://localhost:5000/colors?favorite=false

  • POST: Add a color

    http://localhost:5000/colors

    Example body

    { "name": "Red", "rgb": "FF0000", "favorite": true }

  • PUT: Update a color

    http://localhost:5000/colors

    Example body

    { "id": 1, "name": "Blue", "rgb": "0000FF", "favorite": false }

  • DELETE: Delete a color

    http://localhost:5000/colors

    Example body

    { "id": 1, }
